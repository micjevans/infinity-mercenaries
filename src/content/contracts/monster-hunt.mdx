    <h3>Capturing</h3>
    <p>While the monster is present, players attempt to capture it while it hunts them and tries to hide.</p>
    <ul>
      <li>When the monster spawns, activate the monster.</li>
      <li>Every 4 Orders spent by players, activate the monster. The count does not restart between player turns.</li>
      <li>Once per quarter health lost, after Order resolution, the monster immediately re-enters Hidden Deployment and players must track again. It retains any states it had.</li>
      <li>When the monster goes Unconscious, the shooting player may choose whether additional wounds kill it. It does not re-enter Hidden Deployment.</li>
    </ul>

    <RuleCard eyebrow="State" title="Tag and Bag State">
      <h4>Activation</h4>
      <ul>
        <li>A Trooper has declared Tag and Bag on a Monster.</li>
      </ul>
      <h4>Effects</h4>
      <ul>
        <li>While in Tag and Bag State, the player will move both trooper and Monster simultaneously in Silhouette contact, using the MOV values of the carrying Trooper.</li>
        <li>In this State, the carried Monster <strong>CAN</strong> declare and perform Orders or AROs.</li>
        <li>In Tag and Bag State, the carried Monster does not block LoF.</li>
      </ul>
      <h4>Cancellation</h4>
      <ul>
        <li>Tag and Bag State is automatically cancelled if the Trooper that declared Tag and Bag declares a Skill other than Cautious Movement, Climb, Dodge, Jump, Reset, or a Short Movement Skill against a target that is not the Monster.</li>
        <li>This State is also cancelled when the Trooper that declared Tag and Bag enters any Immobilized or Null State, or any other State that indicates so.</li>
        <li>Tag and Bag State is cancelled when the Trooper that declared Tag and Bag enters a State which replaces the Model with a Marker (Camouflaged, Impersonation, Holoecho...).</li>
        <li>This State is also automatically cancelled if the carried Monster recovers from the Immobilized (IMM-A or IMM-B), or Unconscious State.</li>
        <li>The Trooper that declared Tag and Bag can cancel it voluntarily by spending a Short Skill of an Order, with no Roll required.</li>
      </ul>
